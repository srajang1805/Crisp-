# 📰 Crisp – Lightweight Android News App

Crisp is a fast and minimal Android news application built using **Kotlin**, designed to deliver news efficiently using **RSS feeds**. It focuses on simplicity, speed, and a clean user experience.

---

## ✨ Features

- ⚡ Lightweight and fast
- 📰 Fetches real-time news via RSS feeds
- 📱 Clean and minimal UI
- 🔄 Swipe-to-refresh functionality
- 🌙 Optimized for performance and low memory usage
- 🔗 Open full articles in browser
